14 Each I/O port has 16 individually configurable bits. Many I/O pins share GPIO
15 functionality with a number of alternate functions and must be configured to the
16 alternate function mode if these are to be accessed. A feature is available to
17 remap alternative functions to a limited set of alternative pins in the event
18 of a clash of requirements.
19 
20 The data registers associated with each port for input and output are 32 bit with
21 the upper 16 bits unused. The output buffer must be written as a 32 bit word, but
22 individual bits may be set or reset separately in atomic operations to avoid race
23 conditions during interrupts. Bits may also be individually locked to prevent
24 accidental configuration changes. Once locked the configuration cannot be changed
25 until after the next reset.
26 
27 Each port bit can be configured as analog or digital input, the latter can be
28 floating or pulled up or down. As outputs they can be configured as either
29 push-pull or open drain, digital I/O or alternate function, and with maximum
30 output speeds of 2MHz, 10MHz, or 50MHz.
31 
32 On reset all ports are configured as digital floating input.
33 
34 @section gpio_api_ex Basic GPIO Handling API.
35

80 /** @brief Set GPIO Pin Mode
81 
82 Sets the mode (input/output) and configuration (analog/digitial and
83 open drain/push pull), for a set of GPIO pins on a given GPIO port.
84 
85 @param[in] gpioport Unsigned int32. Port identifier @ref gpio_port_id
86 @param[in] mode Unsigned int8. Pin mode @ref gpio_mode
87 @param[in] cnf Unsigned int8. Pin configuration @ref gpio_cnf
88 @param[in] gpios Unsigned int16. Pin identifiers @ref gpio_pin_id
89  If multiple pins are to be set, use logical OR '|' to separate them.
90 */
91 
92 void gpio_set_mode(u32 gpioport, u8 mode, u8 cnf, u16 gpios)
93 {
94  u16 i, offset = 0;
95  u32 crl = 0, crh = 0, tmp32 = 0;
96 
97  /*
98  * We want to set the config only for the pins mentioned in gpios,
99  * but keeping the others, so read out the actual config first.
100  */
101  crl = GPIO_CRL(gpioport);
102  crh = GPIO_CRH(gpioport);
103 
104  /* Iterate over all bits, use i as the bitnumber. */
105  for (i = 0; i < 16; i++) {
106  /* Only set the config if the bit is set in gpios. */
107  if (!((1 << i) & gpios))
108  continue;
109 
110  /* Calculate bit offset. */
111  offset = (i < 8) ? (i * 4) : ((i - 8) * 4);
112 
113  /* Use tmp32 to either modify crl or crh. */
114  tmp32 = (i < 8) ? crl : crh;
115 
116  /* Modify bits are needed. */
117  tmp32 &= ~(0xf << offset); /* Clear the bits first. */
118  tmp32 |= (mode << offset) | (cnf << (offset + 2));
119 
120  /* Write tmp32 into crl or crh, leave the other unchanged. */
121  crl = (i < 8) ? tmp32 : crl;
122  crh = (i >= 8) ? tmp32 : crh;
123  }
124 
125  GPIO_CRL(gpioport) = crl;
126  GPIO_CRH(gpioport) = crh;
127 }

143 /** @brief Map Alternate Function Port Bits (Main Set)
144 
145 A number of alternate function ports can be remapped to defined alternative
146 port bits to avoid clashes in cases where multiple alternate functions are present.
147 Refer to the datasheets for the particular mapping desired. This provides the main
148 set of remap functionality. See @ref gpio_secondary_remap for a number of lesser used
149 remaps.
150 
151 The AFIO remapping feature is used only with the STM32F10x series.
152 
153 @note The Serial Wire JTAG disable controls allow certain GPIO ports to become available
154 in place of some of the SWJ signals. Full SWJ capability is obtained by setting this to
155 zero. The value of this must be specified for every call to this function as its current
156 value cannot be ascertained from the hardware.
157 
158 @param[in] swjdisable Unsigned int8. Disable parts of the SWJ capability @ref afio_swj_disable.
159 @param[in] maps Unsigned int32. Logical OR of map enable controls from @ref afio_remap,
160  @ref afio_remap_can1, @ref afio_remap_tim3, @ref afio_remap_tim2, @ref afio_remap_tim1,
161  @ref afio_remap_usart3. For connectivity line devices only @ref afio_remap_cld are
162  also available.
163 */
164 void gpio_primary_remap(u32 swjdisable, u32 maps)
165 {
166  AFIO_MAPR |= (swjdisable & AFIO_MAPR_SWJ_MASK) | (maps & 0x1FFFFF);
167 }
