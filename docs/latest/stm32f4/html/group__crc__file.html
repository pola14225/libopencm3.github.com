CRC Calculate of a Block of Data.

Add a block of data to the CRC calculator and return the final result.

Writes data words consecutively to the register, the write operation stalling until the computation of each word is complete.

Parameters
[in]datapUnsigned int32. pointer to an array of 32 bit data words.
[in]sizeint. Size of the array.
